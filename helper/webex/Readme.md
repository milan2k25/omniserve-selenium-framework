Prerequisite / dependencies:
----------------------------

Install python3.8 and pip or pip3

pip installers:
---------------
    pip --upgrade
    pip install requests
    pip install lxml
    
    
note: create a text doc, copy->paste the pip install commands and save them as "pip_installers.sh". 
        [change the file permission 'chmod +x <filename>']
        run the file with sudo.
